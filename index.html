O projeto

CuruKit é uma extensão simples e ágil do projeto CuruWork desenvolvido pelo front-ender Diego Curuma, facilitando as tarefas do dia a dia usando grid do Pure, (mas pode usar o que vc preferir) conta com vários de blocos que podem ser usados em notícias, galerias e etc. Você pode usar e modificar os exemplos sempre que precisar, contribua com o projeto dando um Fork no github.

Como usa?

O kit vem com vários exemplos de blocos e buttons, é só escolher o que for útil para você, usar o html e scss e já era. Você pode ver o exemplo no codepen se preferir.
